Bei allen Projekten immer Git + GitHub nutzen. Ordentlich commiten, pushen, Repos anlegen.

**Why:** Matthias will Best Practices. Versionierung, Nachvollziehbarkeit, und Möglichkeit Sachen wie GitHub Pages zu nutzen.

**How to apply:** Bei jedem Projekt das Dateien erzeugt: Git-Repo anlegen (oder bestehendes nutzen), sinnvolle Commits, auf GitHub pushen. Auch für kleine Sachen wie HTML-Einladungen — das ermöglicht z.B. GitHub Pages Hosting.
